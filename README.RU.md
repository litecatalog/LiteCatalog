## ✨ Особенности
✔️ **Проверка контрольной суммы** — большинство загружаемыеых файл проверяется по SHA1-хэшу, что гарантирует подлинность и целостность скачанного файла.<br>
✔️ **Старые версии** — для ряда программ доступны архивные версии, совместимые с устаревшими Windows.<br>
✔️ **Авто-установка портативных приложений** — архивы (zip, 7z, rar) автоматически распаковываются в папку программ (`C:\Programs\` по умолчанию) с созданием ярлыков на рабочем столе.<br>
✔️ **Тихий режим установки** — установка может выполняться в фоне без диалоговых окон.<br>

## Добавление приложений
Каталог стремится собирать лучшее: популярные, полезные и проверенные временем программы с минимумом дубликатов. Если знаете достойного кандидата - предложите его через [Issues](https://github.com/litecatalog/AppsDB/issues), наличие статьи на Википедии, а также репозиторий с релизма на GitHub будут плюсами.



Если приложение не было добавлено - не расстраивайтесь. Возможно, в каталоге уже есть хороший аналог. Если считаете иначе - аргументируйте, это поможет принять верное решение.

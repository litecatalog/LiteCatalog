## ✨ Особенности
✔️ **Проверка контрольной суммы** - большинство загружаемых файлов проверяется по SHA1-хэшу, что гарантирует подлинность и целостность скачанного файла.<br>
✔️ **Старые версии** - для ряда программ доступны архивные версии, совместимые с устаревшими Windows.<br>
✔️ **Авто-установка портативных приложений** - архивы (zip, 7z, rar) автоматически распаковываются в папку программ (`C:\Programs\` по умолчанию) с созданием ярлыков на рабочем столе.<br>
✔️ **Тихий режим установки** - установка может выполняться в фоне без диалоговых окон.<br>

## 📦 Добавление приложений
### Предложить приложение
Каталог стремится собирать лучшее: популярные, полезные и проверенные временем программы с минимумом дубликатов. Если знаете достойного кандидата - предложите его через [Issues](https://github.com/litecatalog/AppsDB/issues), наличие статьи на Википедии, а также репозиторий с релизма на GitHub будут плюсами.



Если приложение не было добавлено - не расстраивайтесь. Возможно, в каталоге уже есть хороший аналог. Если считаете иначе - аргументируйте, это поможет принять верное решение.

### Добавить самостоятельно
Каждое приложение описывается INI-файлом в папке `Apps\<Категория>\`. После создания можете [предложить Pull Request](https://github.com/litecatalog/AppsDB/pulls) или [предложить готовый файл](https://github.com/litecatalog/AppsDB/issues).

Ключевые поля:

| Поле | Описание |
|------|----------|
| `DownloadURL.x86` / `DownloadURL.x64` | Прямые ссылки на скачивание |
| `SHA1.x86` / `SHA1.x64` | Контрольные суммы файлов |
| `SizeBytes.x86` / `SizeBytes.x64` | Размер файла в байтах |
| `HashCheck` | Проверка хэша (1 - да, 0 - нет) |
| `SilentParams` | Параметры тихой установки, "%PROGRAMS%" заменяется на выбранный каталог для программ |
| `NoArchiveNoInstaller` | Файл является standalone-программой (например, ссылка ведёт сразу на exe программу) |
| `ArchiveDesktopShortcuts` | Ярлыки для портативных приложений (`<Название>=<Путь в архиве до приложения>`, с разделителем `;`) |
| `ArchiveHasInstaller` | В архиве находится инсталлятор (1 - да) |
| `ArchiveInstallerName` | Имя инсталлятора внутри архива |

Для автоматического расчёта хэшей и размеров файлов используйте инструмент отладки. Включить режим отладки можно изменив параметр `Debug` на `1` в конфигурационном файле `Config.ini`. Далее в меню выберите `Посчитать хеши и размеры URL адресов конфига`. Также там можно посмотреть сокращенное название локализации.



Для перевода на другие языки создайте раздел с необходимыми параметрами: `[App.Locale.<locale_name>]`.

Старые версий программ для определенных Windows можно указать в соответствующих разделах: `[App.Windows10]`, `[App.Windows8.1]`, `[App.Windows8]`, `[App.Windows7]`, `[App.WindowsVista]` и `[App.WindowsXP]`.
